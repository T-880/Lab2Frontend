Detta är en enkel webbapplikation som använder en REST-baserad webbtjänst.

Backend är byggd med Node.js, Express och MySQL (MariaDB via XAMPP).

Frontend använder HTML, CSS och JavaScript med Fetch API för att kommunicera med servern.

Sidans funktioner inkluderar att visa, lägga till och ta bort arbetserfarenheter i ett digitalt CV.
